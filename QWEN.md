# bentelega.github.io

Личный сайт-портфолио разработчика, размещённый на GitHub Pages.

## Обзор проекта

Сайт-визитка для разработчика **bentelega**, специализирующегося на создании:
- Веб-приложений (Fullstack)
- Telegram-ботов
- Автоматизации процессов

## Структура проекта

```
bentelega.github.io/
├── index.html              # Заглушка/главная страница
├── portfolio.html          # Основная страница портфолио
├── palette.PNG             # Изображение цветовой палитры
├── deploy.sh               # Скрипт быстрого деплоя
├── CNAME                   # Домен: bentelega.github.io
├── .env                    # ⚠️ Секреты (не коммитить!)
├── .github/
│   └── workflows/
│       └── jekyll-gh-pages.yml  # CI/CD для GitHub Pages
└── static/
    ├── css/
    │   └── portfolio.css   # Стили (896 строк)
    └── js/
        └── portfolio.js    # Интерактивность (260 строк)
```

## Технологии

| Категория | Технологии |
|-----------|------------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Шрифты** | Inter, JetBrains Mono (Google Fonts) |
| **Хостинг** | GitHub Pages |
| **CI/CD** | GitHub Actions (автодеплей при push в main) |

## Запуск и разработка

### Локальный сервер
```bash
python3 -m http.server 8080
```
Открыть: `http://localhost:8080/portfolio.html`

### Деплой
```bash
# Вариант 1: через скрипт
./deploy.sh

# Вариант 2: вручную
git add .
git commit -m "описание изменений"
git push
```

## Структура портфолио

1. **Навигация** — логотип, меню (Главная, Обо мне, Навыки, Контакты)
2. **Герой** — приветствие, имя, описание, кнопки CTA, код-визуал
3. **Обо мне** — текст, статистика (5+ лет, 50+ проектов, 30+ клиентов)
4. **Навыки** — 4 категории: Frontend, Backend, Боты, Инструменты
5. **Контакты** — email, Telegram, GitHub
6. **Футер** — копирайт, ссылки

## Интерактивность (JS)

- Мобильное меню (бургер)
- Плавная прокрутка к якорям
- Анимации при скролле (Intersection Observer)
- Анимация счётчиков статистики
- Эффект печатной машинки для подзаголовка
- Подсветка активной секции в меню
- Уведомления (toast)
