# Portfolio V2 - Maximalist Cyberpunk Design

A bold, distinctive redesign of the developer portfolio with a cyberpunk aesthetic.

## 🎨 Design Concept

**Theme**: Cyberpunk Maximalism
- Vibrant neon color palette (cyan, magenta, gold)
- Playfair Display + Space Grotesk typography
- Animated particles and glowing effects
- Terminal/code aesthetic with modern flair
- Dynamic hover animations and transitions

## 🚀 Features

### Enhanced Visuals
- **Animated particles** in background
- **Pulsing avatar** with gradient glow
- **Terminal code block** with animated header
- **Neon border effects** on hover
- **Smooth parallax scrolling**

### Advanced Animations
- **Staggered scroll animations** with delays
- **Eased counter animations** (quart easing)
- **Terminal code typing effect**
- **Hover transformations** (translate, scale)
- **Active navigation highlighting**

### Responsive Design
- Mobile-first approach
- Adaptive grid layouts
- Touch-friendly interactions
- Optimized for all screen sizes

## 🎯 Key Differences from V1

| Feature | V1 | V2 |
|---------|----|----|
| **Color Scheme** | Purple/Red | Cyberpunk Neon |
| **Typography** | Inter/JetBrains | Playfair/Space Grotesk |
| **Animations** | Basic | Advanced (particles, easing) |
| **Effects** | Simple shadows | Glow, pulse, neon borders |
| **Layout** | Standard grid | Asymmetric, dynamic |
| **Code Style** | Static | Animated terminal |

## 🔧 Technical Highlights

### CSS Features
- CSS variables for theming
- Advanced animations with `@keyframes`
- Backdrop filters and blur effects
- Gradient backgrounds and borders
- Responsive grid layouts

### JavaScript Features
- Intersection Observer for scroll animations
- Performance-optimized animations with `requestAnimationFrame`
- Dynamic particle system
- Enhanced hover effects
- Smooth scroll with parallax
